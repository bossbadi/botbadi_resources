BossBeats is a smarter music bot that can has advanced search and play features. It can stream songs and videos directly from YouTube. It can also stream audio files from other websites. Get started by typing $help.

Here are the main commands:
    $join - Join your voice channel
    $play {Keywords} - Play from YouTube/URL
    $search [Keywords] - Search YouTube

    $pause - Pause music
    $resume/$unpause - Resume music
    $stop - Stop music

    $plist - Show playlist
    $pop {Num} - Remove song from playlist
    $clear - Empty playlist

    $leave/$disconnect - Disconnect from voice channel
    $debug - May fix VC bugs (don't use unless emergency)
    $ping - Check my response time
    $info/$about - Bot information


Every application has bugs, and this bot is no exception. Sometimes the API that is used to stream and play songs does not work, but this is really rare. This bot uses many APIs for music, so if one API doesn't work, it will automatically move on to the next. If all of the APIs don't work, you will receive an error message. Here are some of the fixes that may help you:
    -Try again
    -Disconnect and reconnect yourself and the bot
    -Use the search command to search up your song, then play using the number.
    -Use the debug command
