NumericBoss is a simple utility bot to have in your back pocket. It can do coin flips and dice rolls. But most importantly, it can calculate math problems. Simply pass an arithmetic expression into the calculator command and get your answer ASAP. It can do the basics like addition, subtraction, multiplication, and division. NumericBoss can also calculate higher-level math including exponentiation, roots, logarithms, trigonometry, and some statistics. And to top it off, it supports common unit conversions such as kilometers to miles.

Typing =help is all you need to get started.

Randomness

Command Description Example
flip or coin Flip a two-sided coin ¯\_(ツ)_/¯
roll or dice Roll a six-sided die ¯\_(ツ)_/¯
rand {Start} {End} Generate a random integer =rand 0 100

Math

Command Description Example
c {Problem} or calc {Problem} Calculate a math problem (examples below) =calc abs(log(cbrt(-1)))
graph {Ordered pairs} Graph some ordered-pairs =graph (0 ,2) , ( 3,4), (5, 12 )

Note the braces {} are only placeholders and should not be typed.
